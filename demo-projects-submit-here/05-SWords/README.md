# SWords-women power

**她的言语，就是她的利剑。** *Her Words. Her Weapon.*

## **项目简介**

**SWords** 是一个去中心化女性格言分享平台，聚焦女性创作者的语言表达，结合 Web3 技术记录和展示女性的声音、思想与连结。项目以 “话语即权力” 为核心理念，通过链上记录和社区互动，推动自由表达与文化传承。

> 愿景：让女性的每一句话，都能在时间与链上留下痕迹，成为抵抗遗忘的火种。
> 

### 核心使命

**赋权表达**

- SWords 致力于通过技术与社群支持，保障女性在数字空间中拥有真正的表达自由与言语主权。

**记录与保存**

- 结合 Web3 技术，实现女性格言、观点与思想的不可篡改链上存储，构建属于女性的语录档案馆。

**连接与共鸣**

- 搭建女性创作者之间的思想连结网络，通过言语墙、书籍推荐与互动机制，激发跨时空、跨文化的共鸣与协作。

**文化传承**

- 不仅关注当下，更将女性的思想作为一种文化力量传承给未来，让她们的话语不再被历史抹去。

## 解决方案

### 项目背景

在当今社会，女性的言论依然常常被忽视、打断、误解，甚至被系统性地抹去。无论是在公共讨论、媒体报道，还是学术、艺术与技术等领域，女性声音往往难以成为主流叙事的一部分。她们的表达被轻视为“情绪化”“琐碎”，而非理性、建设性的观点。在网络空间中，女性言说更易遭受性别化攻击与审查，许多女性因此自我审查、沉默或隐退。哪怕是在自我表达最为自由的艺术领域，女性艺术家和创作者的声音也常被边缘化，被归类为“女性主题”，仿佛她们只能代表“性别”而非“人类”整体。这种看不见与被遮蔽，不仅是个人声音的丧失，更是集体经验与认知结构的缺失。

### 🧠 **整体现实效应**

这些模块协同工作，形成一个从 **“她在说话” → “她被记录” → “她被看见” → “她的表达被赋值” → “她影响他人”** 的完整链条，打破女性言语的沉默循环。
解释这套机制从**记录、放大、再现、传播、再分配话语权**等多个层面入手，既有文化深度，也具有技术与传播实践的落点。它们共同构成一个强有力的机制，以**解决女性声音被忽视、不被看见的问题**。

### 1. 👩‍🎨 **女性艺术家 / 作家 / 优秀译者人物档案系统**

📌 **问题回应**：女性在艺术与文化史中长期被边缘化或“匿名化”。

✅ **解决方式**：通过系统地整理、发布和展示女性创作者的档案，包括她们的作品、经历、语录和历史处境，使她们**从“被埋没的名字”变成可被引用、被研究、被记忆的文化主体**。这不仅是**赋予其文化存在权**，更是一次**对知识结构的修复**。

> 她们不仅“存在”，她们也有“姓名”。
> 

只读女作家，也可以很快乐。

我们并不是忽视女性作家的作品，而是在主流媒体与出版机制不断推送男性作家的声音时，被动接受了有限的选择。如果有一个聚焦优秀女性作家与译者的档案系统，让我们能自由筛选、主动发现，我们会发现——只读女性写作者，也可以是轻松而愉悦的体验。因为她们的文字更贴近我们的情感、经验与感知，不会在阅读中被突如其来的“爹味”打断思绪，而是温柔地回应我们。

---

### 2. 👩‍🎨 **让“男人背后的女人”被看见**

📌 **问题回应**：女性在历史中往往被遮蔽在男性成功背后，如女编辑、女伴侣、缪斯、笔友等。

✅ **解决方式**：揭示那些在主流叙事中被隐藏的女性支持者、协作者与真正的贡献者，纠正历史记录中的“性别盲区”。

> 不只是“她帮了他”，而是“她创造了他的一部分”。
> 

---

### 3. 🔗 **资源推介：女性友好组织、优质文章、友情链接**

📌 **问题回应**：女性表达者在社群中缺乏连结，常常“各自为战”。

✅ **解决方式**：通过整理并展示其他女性主义组织、社群、文章与资料库，**建立一个信息互联的生态系统**，让女性不再感到孤立无援。在信息时代中重建“她者之间的网络”，增强横向的文化支持力。

> 看见别人，也让我们更容易被看见。
> 

---

### 4. ✏️ **女性名言数据库**

📌 **问题回应**：女性的话语常被视为“私语”“情绪化”，难以进入“公共语录”体系。

✅ **解决方式**：建立一个多领域女性名言库，让女性思想不再只存在于作品“边角”，而能以独立语句、思维力量、批判性观点进入大众视野。**将“她的话”转化为“权威表达”**，打破男性知识体系对引语的垄断。

> “她说”可以像“他说”那样成为真理的源头。
> 

---

### 5. 🪙 **格言铸造成 NFT（鼓励表达 + 文化资产）**

📌 **问题回应**：女性表达缺乏结构性激励，也易被剽窃、消失在信息流中。

✅ **解决方式**：将女性创作的格言、片段、思想句子铸造成链上 NFT，让“表达”不仅是发声，更成为一种**可被认证、收藏、流通的文化资产**。这既是一种**精神激励**，也是对女性知识劳动的**经济承认**。

> 每一句女性的思想，都值得被珍藏，而非遗忘。
> 

---

### 6. 💌 **邮件订阅模块（每日格言推送）**

📌 **问题回应**：女性表达虽然存在，但缺乏持续曝光渠道，常被新内容冲淡、淹没。

✅ **解决方式**：通过“每日一句”的推送机制，让女性名言/思想片段**进入人们的日常语境中**。不仅建立节奏感，也形成记忆的持久性。推送机制使女性话语成为“文化提醒器”，不断唤起注意、启发思考。

> 不只是一次性表达，而是日常中的持续在场。
> 

### 7. 🎲 **趣味页面（每日一抽——书籍与名言）**

📌 **问题回应**：女性主义表达常常承载沉重议题，容易让用户感到疲惫、距离感较强，缺乏一种“轻盈的入口”和“持续的情绪连接”。

✅ **解决方式**：设置“每日一抽”趣味页面模块，用户每天可点击抽取一条随机女性格言或一本推荐书籍，轻松获取启发、发现新内容。配合动态动效或互动设计，赋予平台一种**日常陪伴感、灵感触发感与情绪疗愈力**，增强用户对女性内容的亲近度与探索欲。同时，鼓励分享与再传播，可将抽到的海报保存转发。

> 有些语言不是为了论战，而是为了温柔地提醒你：你并不孤单。
> 
> 
> 今天的你，或许会被一句话轻轻接住。
> 

---

### 8. 🫂 **女性互助（言论友好墙）**

📌 **问题回应**：在许多网络场域中，女性表达不仅面临打断与忽视，更常遭受性别攻击、冷嘲热讽，导致许多女性在表达前就选择沉默。缺乏一个真正“安全且可被理解”的表达环境。

✅ **解决方式**：构建“言论友好墙”作为平台的公共回应空间，用户可以在墙上自由发布感受、评论、经验碎片，不必完美、不必写作，只需真实。能够强化平台语境的**心理安全感，**形成女性之间的温柔链结，真正打破“她者孤岛”的结构性困境。

> 这里不是论辩场，而是我们一起靠近语言的地方。
> 
> 
> 不是为了争论谁对谁错，而是提醒你：你可以说出口，而我们听得懂。

## 技术框架

| 模块 | 技术/工具 | 描述 |
| --- | --- | --- |
| 前端 UI | [v0.dev](https://v0.dev/) + Tailwind + Shadcn UI | 通过 v0.dev 自动生成页面模板并使用 Tailwind CSS 美化，Shadcn 组件集成交互。 |
| 前端框架 | Vite + React | 使用现代轻量级开发框架构建 SPA 页面。 |
| 状态管理与交互 | wagmi + OneKey Wallet | 实现钱包连接、链上身份绑定、合约交互（支持 OneKey 钱包）。 |
| 后端/数据库 | Supabase (PostgreSQL) | 存储格言内容、作者、评论等信息；也提供 RESTful API 接口。 |
| 区块链智能合约 | Solidity（部署至 Ethereum Sepolia 测试链） | 提交格言与评论上链，绑定钱包地址，合约事件驱动交互。 |
| 部署平台 | Vercel | 项目使用 Vercel 持续部署，自动构建与预览环境。 |
